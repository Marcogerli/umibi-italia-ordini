Base tecnica

Software installati sul computer

Visual Studio Code

Editor principale usato per scrittura codice, modifica file, test e manutenzione del progetto.

Node.js

Necessario per avviare l’app in locale e generare la build finale. Include anche npm per le dipendenze.

Browser web

Usato per provare l’app durante sviluppo e pubblicazione. Esempi: Chrome ed Edge.

Questi strumenti costituiscono la base minima per lavorare in autonomia sul progetto e generare nuove versioni dell’app.

Stack del progetto

Librerie e tecnologie già utilizzate

Frontend

  • React per costruire l’interfaccia
  • React DOM per la visualizzazione nel browser
  • TypeScript per un codice più controllato
  • Vite per avvio rapido e build
  • vite-plugin-pwa per la versione installabile

Funzioni principali

  • login e registrazione utenti
  • gestione ordini e storico
  • pubblicazione online della demo
  • invio email automatiche di conferma
  • export ordini in Excel multi-foglio
  • La struttura attuale è già adeguata per una web app dimostrativa e operativa.
  • La base può essere estesa in futuro con database cloud e pubblicazione mobile.

Servizi online

Servizi configurati per il funzionamento dell’app

Clerk

Gestisce autenticazione, registrazione, sessioni e recupero password.

EmailJS

Invia l’ordine all’azienda e la conferma al cliente tramite template email.

Netlify

Pubblica online la demo e genera un link pubblico accessibile via browser.

L’app è quindi già supportata da un ecosistema completo per autenticazione, comunicazione, pubblicazione online e installazione mobile.

Admin, export Excel e PWA già integrati

Pannello admin

  • visualizzazione di tutti gli ordini
  • filtro per cliente e numero ordine
  • cambio stato ordine in tempo reale
  • statistiche con fatturato, clienti e prodotti

Export Excel

  • file XLSX multi-foglio
  • foglio ordini completo
  • foglio statistiche
  • top clienti e prodotti più ordinati

PWA e grafica

  • app installabile da smartphone
  • icone aggiornate con logo Umibi
  • nuovi bottoni e layout più chiaro
  • grafica responsive per mobile e desktop
Queste funzioni rappresentano la versione più recente dell’app e rendono il progetto più completo, professionale e adatto a un uso reale.
